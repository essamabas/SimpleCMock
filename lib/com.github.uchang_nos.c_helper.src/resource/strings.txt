# キーには空白文字を含めてはいけません

行頭から書き始めるのは分かりにくい。%d個分インデントすべき。=\
    行頭から書き始めるのは分かりにくいため、スペース %d 個分インデントすべきです。

インデントが乱れている。%d個分インデントすべき。=\
    インデントが乱れています。スペース %d 個分インデントすべきです。

インデントに用いる文字は統一すべき。前方では%sここでは%sが用いられている。=\
    インデントに用いる文字は統一すべきです。前方では %s が、ここでは %s が用いられています。

関数の定義は行頭から書き始めると綺麗=\
    関数の定義は行頭から書き始めると綺麗です。

char型配列の1つの要素に文字列を格納できない。=\
    char型配列の1つの要素に文字列を格納できません。 strcpy を使うことを検討してください。

char型変数に文字列を格納できない。=\
    char型変数に文字列を格納できません。 char型配列を検討してください。

式%sが1文字を表しているなら%%cで表示可能。=\
    式 %s が1つの文字を表しているなら %%c を使えば表示可能です。

整数からポインタ型へのキャストは危険。=\
    整数からポインタ型へのキャストは危険です。

printfに引数が1つも指定されていない。=\
    printf 関数に引数が1つも指定されていません。

printf(hello)などと書く。=\
    printf(\"hello, world\\n\") などと書きます。

printfの第一引数は文字列定数であるべき。=\
    printf 関数の第一引数に渡す文字列は、変数ではなく文字列定数であることが推奨されます。

printfの第一引数には書式文字列を渡す。=\
    printf 関数の第一引数には書式指定文字列を渡します。

printfの引数が足りない。=\
    printf 関数の書式指定文字列の%変換に用いる引数の数が足りません。

printfの引数が多すぎる。=\
    printf 関数の書式指定文字列の%変換に対し、引数が多すぎます。

関数の引数は式でなければならない。=\
    C89 (ANSI-C) では関数の引数は「式」でなければなりません。

符号なし整数の表示は%%u,%%xなど。=\
    符号なし整数の表示には %%u, %%x などが使えます。

long符号なし整数の表示は%%lu,%%lxなど。=\
    long 符号なし整数の表示には %%lu, %%lx などが使えます。

longlong符号なし整数の表示は%%llu,%%llxなど。=\
    long long 符号なし整数の表示には %%llu, %%llx などが使えます。

符号付き整数の表示は%%d=\
    符号付き整数の表示には %%d が使えます。

long符号付き整数の表示は%%ld=\
    long 符号付き整数の表示には %%ld が使えます。

longlong符号付き整数の表示は%%lld=\
    long long 符号付き整数の表示には %%lld が使えます。

浮動小数点数の表示は%%fなど=\
    浮動小数点数の表示には %%f, %%e, %%g などが使えます。

文字列の表示は%%s=\
    文字列は %%s で表示できます。

ポインタ値の表示は%%p=\
    ポインタ値は %%p で表示できます。

引数は%sだが%%%cは%sを期待している。=\
    引数は%sですが %%%c は%sを期待しています。

引数と%%変換の型が合わない。=\
    引数と %%変換の型が合いません。

引数型と%%変換のサイズが合わない。=\
    引数の型のサイズと %%変換の型のサイズが合いません。

文字と文字列は比較できない=\
    文字と文字列は比較できません。

1文字を表すには\"ではなく\'で囲む=\
    1文字を表すには\"ではなく\'で囲みます。

ヘッダファイルには実体を定義すべきではない=\
    ヘッダファイルには実体を定義すべきではありません。

extern宣言する=\
    extern宣言を用いてください。

ヘッダファイルにプロトタイプ宣言、ソースファイルに実体を書く=\
    ヘッダファイル(.h)にはプロトタイプ宣言を、ソースファイル(.c)には定義を書きます。

識別子%sが重複している=\
    識別子%sが重複しています。

メモリリークする=\
    メモリリークが発生します。

メモリリークする可能性がある=\
    メモリリークが発生する可能性があります。

ポインタを渡す必要がある=\
    ポインタを渡す必要があります。

&を付けて変数へのポインタを取得する(&%s)=\
    &を付けて変数へのポインタを取得します。(&%s)

%s.hをインクルードする=\
    %s.h を #include してください。

%sは宣言されていない関数名=\
    %s は宣言されていない関数名です。
